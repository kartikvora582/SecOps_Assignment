CVE Database Web App
This is a simple web application built with React.js to manage a CVE (Common Vulnerabilities and Exposures) database.

Usage
Download the ZIP file containing the project from the provided link.
Extract the contents of the ZIP file to your desired location on your local machine.
Dependencies
To run this project locally, you need to have the following dependencies installed:

Node.js
npm (Node Package Manager)
Setup
Navigate to the extracted project directory in your terminal or command prompt.

Install the dependencies using npm:
npm install

Running the Application
Once you have installed the dependencies, you can run the application locally using the following command:
npm start

This will start the development server, and you should be able to access the application at http://localhost:3000 in your web browser.

Usage
The application allows you to view, add, edit, and delete CVE records.
You can filter the CVE records based on CVE-ID, Severity, CVSS, Affected Packages, and CWE-ID.
To add a new CVE record, click on the "Add New CVE" button, fill out the form in the modal, and click "Save".
To edit an existing CVE record, click on the "Edit" button next to the record, make your changes in the modal, and click "Save".
To delete a CVE record, click on the "Delete" button next to the record.
